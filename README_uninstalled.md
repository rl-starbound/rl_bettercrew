# Starbound mod: Better Crew Uninstalled

[Better Crew & NPC Behavior](README.md) adds scripts to certain entities in your worlds, and simply removing it may cause your worlds to crash. If you no longer want to use that mod, you must remove it and replace it with *Better Crew Uninstalled*, which provides a minimal set of assets that allow the safe removal of *Better Crew & NPC Behavior*.

Upon replacing *Better Crew & NPC Behavior* with this mod, you should notice your NPCs behaving as they did in the base game.

If you never installed *Better Crew & NPC Behavior*, there is no reason for you to install this mod.

## Compatibility

This mod should be compatible with every mod.

## Reinstallation of Better Crew

If you want to reinstall *Better Crew & NPC Behavior*, simply remove this mod and replace it with that one.

## Uninstallation

This mod cannot be removed safely. However, it provides only the minimal set of assets needed to remove *Better Crew & NPC Behavior* safely, so you should not notice its existence.
