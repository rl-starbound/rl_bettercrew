# Starbound mod: Better Crew - No Quickbar

The *Better Crew: No Quickbar* mod is an extension to [Better Crew & NPC Behavior](README.md), and requires that mod. It enables the [crew rallying](user-guide.md#crew-rallying) functionality of *Better Crew* for users who do not have the *Stardust Core Lite* or *Stardust Core* mods installed. Users who have *Stardust Core Lite* or *Stardust Core* do not need this mod.

When this mod is installed, an object called the Crew Communicator is awarded to players upon completion of the ship repair quest. Alternatively, it can be purchased from Penguin Pete at the Outpost. Equipping it and using it will summon non-following crew members on the same (non-ship) world as the player.

## Uninstallation

This mod may be removed at any time. If a player obtained a Crew Communicator while this mod was installed, they will retain that item, but no new copies can be obtained without this mod.
